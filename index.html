Tic-Tac-Toe

I am currently taking a class on AI, and wrote the code to make this game work! The idea is that the (human) player will never be able to win against the AI, a tie is best case scenario for the human. You can try playing it yourself by checking it out in my git repository here. As a disclaimer, I wrote the functions that make this game work, the python script that runs the game was provided by the CS50 AI course on EdX.

Light Curve of V* NR Lyr, an RR Lyrae star

Observational data was obtained from my TF. The top panel shows a portion of the field with some of the reference stars and the science star circled. The bottom panel is the light curve. The plotted points are the photometry data collected with our observations and the curve is Kepler data for the same star. The fit is remarkably close. M67

Raw images were taken at Whipple Observatory in Arizona with a class, this image was created using ImageJ and the fits files in i,g, and r filters. Notice that the image is divided into 4 different sections during the read out process from the CCD, each section has its own biases and flats. While the distiction may seem large in the final image, the pixel values don't vary by more than .001 between the regions. Click here to view the image full size.
